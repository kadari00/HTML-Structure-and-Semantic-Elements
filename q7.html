✅ Built-in HTML5 Validation

The form above uses required, min/max (date), and type constraints. Try submitting with an empty date or out-of-range value — the browser will show a validation message automatically.

  • type="date" + required + min/max
  • type="range" with min="0" max="100"
  • type="color" (always valid, but shows a color picker)

✔ All inputs are inside a <form> with validation enabled.
